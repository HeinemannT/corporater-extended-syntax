A specialized addon for **Corporater Extended Code**, designed to mimic the standard Config Studio editor with superior highlighting, safety checks, and architectural tools.

## **Features**
**Semantic Highlighting:** More highlighting support and a bit more distinguishing of different elements.
**Light mode:** Aligned to Configuration studio
**Dark Mode:** A more custom theme.

**Error Handling**: Linting for pitfalls, currently:
    - Flags unsupported IF NOT condition usage.  
    - Warns about unwrapped conditional assignments (v := IF...).

**Productivity Snippets**

- IF-transaction: Safe transactional blocks.  
- select-std: SELECT query pattern.  
- json-string & json-list: Quick, multi-line JSON structures.  
- table-addcolumn: Quick table definitions.
